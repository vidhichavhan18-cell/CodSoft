# CodSoft
To-Do List Application
Project Description

The To-Do List Application is a simple command-line Python program that helps users manage their daily tasks efficiently. Users can add, update, delete, and view tasks through an interactive menu-driven interface.

Features
Add new tasks
Update existing tasks
Delete tasks
View all tasks
User-friendly menu interface
Input validation for incorrect menu choices
Technologies Used
Python 3
Lists for task storage
Loops and conditional statements
Exception handling
